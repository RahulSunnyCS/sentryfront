===========================================
IMPORTANT NOTES:
===========================================

1. NEXTAUTH_URL: Change "vibesafe" to your actual project name on Vercel
   Example: https://your-project-name.vercel.app

2. DATABASE_URL: Will be added automatically when you create Vercel Postgres database
   - In Vercel import screen, click "Add Storage" → "Postgres" → "Create"
   - Vercel will inject DATABASE_URL automatically

3. After deployment, you need to run database migrations:
   vercel env pull .env.production
   npx prisma migrate deploy

4. Update OAuth callbacks after deployment:
   - GitHub: https://your-app.vercel.app/api/auth/callback/github
   - Google: https://your-app.vercel.app/api/auth/callback/google
